## About

ylearner makes professional programming education accessible to everyone — no paywall, no signup required. Every course covers topics from beginner fundamentals through advanced concepts, with real code examples, interview preparation, and hands-on projects.

## Tech Stack

- Static HTML/CSS/JavaScript (no build step)
- Hosted via GitHub Pages at `ylearner.org`
- Google AdSense for funding
